# Ducks Can Drive ReBinds- Custom Controls Mod

![Mod Version](https://img.shields.io/badge/version-1.0.0-brightgreen)
![MelonLoader](https://img.shields.io/badge/MelonLoader-0.7.1+-blue)

I Really wanted to change a Key so I made that happen!
A MelonLoader mod for the game **Ducks Can Drive** that adds a fully customizable key binding system.

---

## 📥 Installation

1. **Install MelonLoader**  
   If you haven't already, follow the [MelonLoader installation guide](https://melonwiki.xyz/).

2. **Download the mod**  
   Grab the latest `DCDCustomControls.dll` from the [Releases](https://github.com/yourusername/DCDCustomControls/releases) page.

3. **Place the DLL**  
   Copy the `.dll` file into your game's `Mods` folder.  
   Typical path: `...\steamapps\common\Ducks Can Drive\Mods\`

4. **Launch the game**  
   The mod will automatically initialize. You should see a "Controls" button in the Settings menu (accessible from the main menu).

## 🎮 How to Use

1.You Know how to use a Controls menu, Right?

### Default Key Bindings

| Action  | Default Key |
|---------|-------------|
| Forward | W           |
| Backward| S           |
| Left    | A           |
| Right   | D           |
| Drift   | Space       |
| Honk    | H           |
| Restart | R           |

> **Note:** The Restart key does not yet Change buttons – it only shows a message. The binding is saved for future updates.

---

## ⚙️ Configuration

Bindings are stored in MelonLoader's preferences file. You can also manually edit them by opening `UserData/MelonPreferences.cfg` and looking for the `[KeyBindings]` section.
